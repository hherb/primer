## Patterns to reuse, not reinvent

New from this session:
- **Split an over-500-line module into a directory module (`foo/mod.rs` + `foo/tests.rs`), not a `#[path]` sibling**, to match the in-crate `qnn::backend` precedent. Use `git mv foo.rs foo/mod.rs` first so blame stays on the implementation, then carve the `#[cfg(test)] mod tests { … }` body out into `foo/tests.rs` (de-indent by 4) and replace it with `#[cfg(test)] mod tests;`. `super` / `super::super` paths in the moved tests resolve unchanged because the module *nesting* is preserved — only the file boundary moved. `cargo fmt -p <crate>` after the carve normalises any indentation drift.
- **For a mechanical file split, prefer `git mv` + `sed`/`head` carving over retyping** — a 785-line hand-retype invites transcription errors a behaviour-preserving move must not have. Verify by test count (30 here), not just "it compiles".

Carried forward (prior handoffs): model a multi-file asset bundle as N single-file `kind`s to reuse single-file download infra; locale-independent asset cache when one model serves all locales; a pure mapping helper for a config gate keeps a Tauri-command decision unit-testable; `--features speech` (feature-gated) clippy is invisible to `--workspace` clippy — run it explicitly; decouple orthogonal backend axes by injecting one into the other's builder (Option A); feature-gated constructor with a rebuild hint; GUI owns mirror config enums when the engine crate is an optional dep; one-shot config migration must be applied at the View layer too; clap `required_if_eq` is blind to default values (ArgGroup + runtime validator); free-form GUI text → parse in pure Rust; a hidden form field must still be sent by `gather()` if its DTO field is mandatory; stream-spanning text transforms need a stateful filter; `generate()` aggregates `generate_stream()`; user-facing strings route through `render_inference_error`; run cargo from `src/` so the 1.88 pin is honored.
