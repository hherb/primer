## Patterns to reuse, not reinvent

New from this session:
- **Tauri already git-ignores `jniLibs/**/*.so`** (`gen/android/app/.gitignore`). So proprietary native libs staged into `jniLibs/arm64-v8a/` bundle into the APK at build time but never get committed — exactly the mechanism you want for licence-gated blobs. Commit a sha256-pinned staging README beside them, not the blobs.
- **`primer-qnn-sys` dlopens `libGenie.so` at RUNTIME** (not link-time), so you can compile-check the whole `qnn` feature for `aarch64-android` with NO QAIRT libs on disk. Do the compile-check first; stage the libs only for the actual APK packaging.
- **`--features` is additive to defaults** — `cargo-tauri android build … -- --no-default-features --features qnn` is required to stay BM25-only (issue #157) while adding qnn. Dropping `--no-default-features` would drag the device-unverified aarch64-android ort download back in.
- **`libcdsprpc.so` (FastRPC client) is a device system/vendor lib** (`/vendor/lib64/`), resolved at runtime — do NOT bundle it into the APK (the brief's sub-project-2 note listed it, but bundling a vendor FastRPC stub can break the DSP handshake).
- **A squash-merged feature branch isn't a literal ancestor of `main`** — `git branch -d` refuses it; verify trees match (`git diff <branch> origin/main` empty) then `git branch -D`.

Carried forward (prior handoffs): a new inference-routing/resilience behavior is a decorator over `Arc<dyn InferenceBackend>`, built by `build_main_backend`, `name()` → primary, fall over ONLY pre-stream; routing/scoring POLICY pure in `primer-core`, MECHANISM in `primer-inference`; over-500-lines-only-because-of-tests → `foo/{mod,tests}.rs`; mirroring a CLI feature into the GUI is a 3-struct (`Config`/`View`/`Update`) + wiring + frontend pattern (the `Update` DTO has no `#[serde(default)]`); run cargo from `src/` with `+1.88`; docs-only PRs are CI-path-ignored (#168) → admin-merge/direct-push. Android host facts: JDK 21 = Android Studio's JBR for `JAVA_HOME`; the `$ANDROID_HOME/ndk/29.0.14206865 → /opt/homebrew/share/android-ndk` symlink Tauri 2.11 needs.
