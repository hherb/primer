# Klexikon (German children's wiki) whitelist for the Primer.
#
# Each line is the canonical Klexikon article title. Lines starting
# with `#` are ignored. Titles are case-sensitive and must match
# Klexikon exactly (the API returns "missingtitle" for typos).
#
# Hand-curated set covering the same five clusters the English seed
# corpus uses: space, body, how-things-work, life, earth/weather.
# Some titles redirect to a canonical form on Klexikon — sometimes a
# plural (`Säugetier` -> `Säugetiere`, `Hund` -> `Hunde`), sometimes
# a synonym or broader topic (`Berg` -> `Gebirge`, `Universum` ->
# `Weltall`, `Atom` -> `Atome und Moleküle`). The script follows
# the redirect transparently and stores the canonical title, so the
# JSONL id is computed from the resolved title, not the input line.
# Two distinct input titles whose redirects collapse to the same
# canonical (e.g. `Atom` and `Molekül` both resolve to `Atome und
# Moleküle`) would produce duplicate ids — the ingest now hard-fails
# in that case. To expand: add lines and re-run
# `python3 simple_wikipedia.py --language de`.
#

# Source: https://klexikon.zum.de/
